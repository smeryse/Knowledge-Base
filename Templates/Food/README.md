# Food Templates

Эта папка хранит только Templater-команды для food-проекта.

- `Templates/Food/*.md` = входные точки, которые нужно запускать через Templater
- `Projects/Еда/Templates/*.md` = проектные шаблоны сущностей и справочные формы
- `Templates/scripts/*.js` = пользовательские скрипты Templater

Так общие шаблоны хранятся отдельно от food-команд, а настройки Templater остаются простыми: `templates_folder = Templates`.
