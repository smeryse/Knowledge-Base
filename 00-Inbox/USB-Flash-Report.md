## 8. Рекомендации

- ⚠ Флешка работает в режиме **USB 2.0** — для большей скорости используйте порты USB 3.0/3.1
- ✅ Файловая система **exFAT** подходит для файлов >4GB
- 🔒 Рекомендуется резервное копирование `passport_secure.7z`
- ⚠ При извлечении: `umount /media/smeryse/s`

---

## 9. Команды для управления

```bash
# Просмотр информации
lsblk -o NAME,SIZE,TYPE,MOUNTPOINT,LABEL /dev/sda

# Проверка места
df -h /media/smeryse/s

# Безопасное извлечение
umount /media/smeryse/s

# Проверка ошибок
fsck.exfat /dev/sda
```
