### 🧠 Контекст

Вузовский Wi-Fi использует captive portal:

- первый вход → номер телефона + SMS
- последующие → ручное нажатие «подключиться»
- интернет фактически доступен сразу после авторизации (таймер — фиктивный)

Проблема: **лишние действия при каждом подключении**, особенно на телефоне и ноутбуке.

---

### 🎯 Цель

Минимизировать или полностью убрать ручное взаимодействие при подключении к сети.

---

### 🔍 Что изучить

#### 1. Идентификация устройства

- Как сеть определяет устройство:
    - MAC-адрес
    - cookies / session
- Включен ли **Random MAC**:
    - Android / iOS / Windows / Linux
    - влияет ли на сброс сессии

---

#### 2. Поведение captive portal

- Когда сбрасывается сессия:
    - по времени?
    - при смене точки доступа?
- Какие запросы триггерят портал:
    - HTTP vs HTTPS
    - DNS hijacking

---

#### 3. Возможность автоматизации

##### 📱 Телефон

- Android:
    - Tasker / MacroDroid:
        - авто-открытие страницы портала
        - авто-нажатие кнопки (через AutoInput)
    - проверка URL портала
- iOS:
    - ограничения системы (скорее всего сложнее)

##### 💻 ПК

- Автоматическое открытие портала:
    - через скрипт (например, curl + запрос авторизации)
- Проверка:
    - можно ли повторить запрос кнопки программно
    - анализ через DevTools (Network → что происходит при нажатии)

---

#### 4. Реверс портала

- Открыть DevTools → Network
- Нажать «подключиться»
- Найти:
    - endpoint (POST/GET запрос)
    - параметры (token, session id)
- Попробовать:
    - повторить запрос вручную (curl / Postman)

---

### ⚙️ Возможные решения

#### Вариант 1 — стабилизация сессии

- отключить random MAC
- закрепить устройство в системе

#### Вариант 2 — полуавтомат

- скрипт / приложение открывает портал автоматически
- 1 клик вместо ручного поиска страницы

#### Вариант 3 — полная автоматизация

- воспроизведение запроса авторизации
- авто-выполнение при подключении к Wi-Fi

### ⚠️ Ограничения

- Возможны ограничения со стороны сети (token, CSRF, cookies)
- Автоматизация может ломаться при изменении портала
- Юридические/правила вуза (не лезть слишком глубоко в обход)
