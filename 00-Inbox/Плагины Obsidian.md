# 🧩 Плагины Obsidian

## Основные плагины

| Плагин | Что делает | Ссылка |
|--------|------------|--------|
| **Dataview** | Таблицы, списки, запросы к базе знаний | `obsidian://show-plugin?id=dataview` |
| **Obsidian Charts** | Графики и диаграммы | `obsidian://show-plugin?id=obsidian-charts` |
| **Kanban** | Канбан-доски | `obsidian://show-plugin?id=kanban` |
| **Projects** | Таблицы как в Notion | `obsidian://show-plugin?id=obsidian-projects` |
| **Banners** | Красивые заголовки с изображениями | `obsidian://show-plugin?id=obsidian-banners` |
| **Various Complements** | Автодополнение тегов, ссылок, эмодзи | `obsidian://show-plugin?id=obsidian-various-complements` |
| **Tag Wrangler** | Управление тегами (переименование, объединение) | `obsidian://show-plugin?id=tag-wrangler` |
| **Admonitions** | Красивые информационные блоки | `obsidian://show-plugin?id=admonition` |
| **DataviewJS** | JavaScript-запросы к базе | (входит в Dataview) |

## Для чего какой плагин

### 📊 Визуализация данных
- **Dataview** — таблицы и списки
- **Obsidian Charts** — графики, диаграммы
- **Projects** — интерактивные таблицы

### 📋 Организация
- **Kanban** — доски для задач
- **Tag Wrangler** — работа с тегами

### 🎨 Красота
- **Banners** — баннеры на страницы
- **Admonitions** — цветные блоки

### ⚡ Удобство
- **Various Complements** — автодополнение

---

## Как установить

1. **Settings** → **Community plugins** → **Browse**
2. Ввести название плагина в поиск
3. **Install** → **Enable**

Или кликнуть по ссылке в таблице выше (откроется в Obsidian).
