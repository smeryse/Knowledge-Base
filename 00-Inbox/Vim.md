## Режимы Vim

| Режим                    | Назначение                                     | Вход из нормального режима                      | Выход из режима                     |
| ------------------------ | ---------------------------------------------- | ----------------------------------------------- | ----------------------------------- |
| Normal (Нормальный)      | Навигация и команды                            | По умолчанию при запуске                        | —                                   |
| Insert (Вставка)         | Ввод текста                                    | `i`, `a`, `o` и др.                             | `Esc`                               |
| Visual (Визуальный)      | Выделение текста                               | `v` (символы), `V` (строки), `Ctrl+v` (блочное) | `Esc`                               |
| Command-line (Командный) | Ввод команд (с `:`)                            | `:`                                             | `Enter` (выполнить), `Esc` (отмена) |
| Replace (Замена)         | Замена символов при вводе                      | `R`                                             | `Esc`                               |
| Select (Выбор)           | Выделение с поведением как в других редакторах | (редко используется)                            | `Esc`                               |

## Шпаргалка по редактированию файла конфигурации .vimrc

Ниже приведены самые полезные настройки, которые можно добавить в ваш файл `~/.vimrc`.  
Они ускорят работу, сделают Vim более удобным и помогут избежать типичных ошибок.

| Настройка | Что делает | Пример |
| --------- | ---------- | ------ |
| **Путь к файлу** | Указывает путь к конфигурации | `set runtimepath+=~/.vim` |
| **Автосохранение** | Сохраняет файл при выходе из режима Insert | `autocmd InsertLeave * silent! w` |
| **Подсветка синтаксиса** | Включает подсветку для всех файлов | `syntax on` |
| **Показ номеров строк** | Показывает номера строк слева | `set number` |
| **Автодополнение** | Включает автокомплит в Insert режиме | `set completeopt=menuone,noinsert,noselect` |
| **Подсветка текущей строки** | Делает цвет фона текущей строки более ярким | `highlight CursorLine ctermbg=236 guibg=#2e2e2e`<br>`set cursorline` |
| **Переход к следующему/предыдущему файлу в буфере** | Быстрый переход по открытым файлам | `nnoremap <C-Right> :bnext<CR>`<br>`nnoremap <C-Left> :bprev<CR>` |
| **Сокращения для команд** | Создаёт пользовательские сокращения | `cmap w!! %!sudo tee > /dev/null` (сохранить как root) |
| **Переопределение клавиш** | Переназначает горячие клавиши | `nnoremap <C-s> :w<CR>` (Ctrl+S сохраняет) |
| **Включение подсветки ошибок в терминале** | Делает ошибки более заметными | `set errorformat=%f:%l:\ %m` |
| **Переход к последнему изменению** | Быстрый переход к месту последнего изменения | `nnoremap <C-g> :normal! g;<CR>` |
