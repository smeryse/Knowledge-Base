## Основные сочетания клавиш
- Ctrl + U — удалить строку от курсора до начала
- Ctrl + W — удалить слово слева от курсора
- Ctrl + C — прервать выполнение программы

## Оболочки
- zsh
- bash

## Стандартные потоки
- 0 — stdin (ввод)
- 1 — stdout (вывод)
- 2 — stderr (вывод ошибок)
- > — перенаправить вывод (перезаписать файл)
- >> — дописать в файл

## Работа с файлами и каталогами
- rm — удалить файл или каталог
- rm -r — удалить каталог с файлами рекурсивно
- rmdir — удалить пустой каталог
- cp <источник> <назначение> — копировать файл (например, `cp rainbow.cpp rainbow2.cpp`)
- mv <источник> <назначение> — переместить или переименовать файл/каталог

## Просмотр и управление файлами
- ls -a — показать все файлы, включая скрытые
- ls -l — показать подробную информацию о файлах
- ls -la — объединение флагов -l и -a

### Разбор вывода ls -l
Пример:  
`-rw-rw-r-- 1 smeryse smeryse 3067606 янв 19 15:03 yamusic-tui-vendor.tar.gz`  
- `-rw-rw-r--` — права доступа  
- `1` — количество жестких ссылок  
- `smeryse smeryse` — владелец и группа  
- `3067606` — размер файла в байтах  
- `янв 19 15:03` — дата и время последнего изменения  
- `yamusic-tui-vendor.tar.gz` — имя файла

## Права доступа
- chmod 777 — полный доступ (чтение, запись, выполнение) для всех  
- chmod 755 — полный доступ для владельца, чтение и выполнение для группы и остальных  
- Права представлены в виде суммы:  
  - r (read) = 4  
  - w (write) = 2  
  - x (execute) = 1  
- Пример: 7 = 4 + 2 + 1 (чтение + запись + выполнение)

## Управление владельцами
- chown — изменить владельца и группу файла

## Архивирование
- tar -cvf archive.tar directory/ — создать архив из каталога  
- tar -xvf archive.tar — распаковать архив

## Поиск файлов
- find /path/to/search -name "filename" — найти файл по имени

## Привилегии
