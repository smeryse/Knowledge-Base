Разговор с Полупановой по поводу курсовых

Общая картина: с этого года помимо обыкновенных курсовых, можно делать курсовые проекты (упор идёт в практикоориентированность, более углубленный материал, каждый берёт свою задачу в рамках проекта, вкр будет индивидуальной, но также может быть в рамках одного проекта), научрук может быть с любой кафедры (кмм под вопросом)

Основные темы квт (можно не по теме, если препод одобрит): 
Лапина: нейросети, гиперграфы
Руденко: нейросети, распознавание образов
Приходько: компьютерные сети, мобильные приложения, фронтэнд, бекэнд
Вишняков: семантическая интерпретация (анализ текстов,  аналогичные задачи)
Полупанова: биоинспирированные алгоритмы для решения алгоритмических NP-трудных задач, есть геометрическое моделирование
