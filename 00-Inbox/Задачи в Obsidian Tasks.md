| Команда                            | Описание и примеры                                                                                                                       |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `done` / `not done`                | Фильтр по статусу выполнения                                                                                                             |
| `due <условие>`                    | Фильтр по дедлайну. Примеры: `today`, `tomorrow`, `2026-05-06`, `after 2026-05-01`, `before tomorrow`. Аналогично для `created` и `done` |
| `path includes/excludes <путь>`    | Фильтр по расположению файла                                                                                                             |
| `filename includes/excludes <имя>` | Фильтр по имени файла                                                                                                                    |
| `tags includes/excludes #тег`      | Фильтр по тегам задачи                                                                                                                   |
| `priority <уровень>`               | Фильтрация по приоритету: `lowest`, `low`, `medium`, `high`, `highest`                                                                   |
| `group by <поле>`                  | Группировка: `due`, `path`, `filename`, `tags`, `priority`, `created`, `status`                                                          |
| `sort by <поле>`                   | Сортировка: `due`, `priority`, `created`, `path`, `filename`, `description`                                                              |
| `limit <число>`                    | Ограничение количества выводимых задач                                                                                                   |
| `hide <элемент>`                   | Скрытие элементов: `backlink`, `edit button`, `task count`, `priority`, `created date`, `done date`, `due date`, `tags`                  |

| Порядок | Тип команды       | Примеры                                                                   | Назначение в цепочке                                             |
| ------- | ----------------- | ------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| 1       | Фильтры           | `not done`, `due today`, `path includes Рабочие`, `tags includes #проект` | Первичный отбор. Уменьшает пул задач перед дальнейшей обработкой |
| 2       | Группировка       | `group by filename`, `group by tags`                                      | Разбивает отфильтрованные задачи на блоки                        |
| 3       | Сортировка        | `sort by priority`, `sort by due`                                         | Упорядочивает элементы внутри каждой группы                      |
| 4       | Лимит             | `limit 50`                                                                | Ограничивает итоговое количество строк после сортировки          |
| 5       | Скрытие элементов | `hide backlink`, `hide task count`, `hide edit button`                    | Визуальные настройки, не влияют на логику выборки                |

| Заголовок               | Команды                                                                                                         | Назначение                                                                                                                           |
| ----------------------- | --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **Задачи на сегодня**   | `not done`<br>`due today`<br>`group by filename`<br>`sort by priority`<br>`hide backlink`<br>`hide edit button` | Выводит невыполненные задачи на текущую дату. Группирует по файлам, сортирует по приоритету, скрывает ссылки и кнопку редактирования |
| **Все активные задачи** | `not done`<br>`path includes Рабочие`<br>`group by tags`<br>`sort by created`<br>`limit 100`                    | Показывает задачи из папки «Рабочие», группирует по тегам, сортирует по дате создания. Ограничивает вывод 100 строками               |
| **Просроченные**        | `not done`<br>`due before today`<br>`sort by due`<br>`hide task count`                                          | Фильтрует задачи с истёкшим сроком. Сортирует от самых старых дедлайнов, скрывает счётчик задач                                      |
