## 🔗 Быстрый выбор типа графика

| Цель | Рекомендуемый тип |
|------|------------------|
| Сравнение категорий | `bar`, `column` |
| Тренд во времени | `line`, `area` |
| Доли от целого | `pie`, `doughnut` |
| Профиль / баланс параметров | `radar` |
| Распределение / частота | `histogram`, `polarArea` |
| Корреляция двух переменных | `scatter`, `bubble` |
| Иерархия / структура | `treemap`, `sunburst` |
| Процесс / этапы | `flowchart`, `gantt` |

## 🧩 Синтаксис плагинов

### Плагин **Charts** (phibr0) — Chart.js
Блок: ` ```chart `
Формат: YAML-упрощённый

```yaml
type: bar
labels: [Янв, Фев, Мар]
series:
  - title: Доход
    data: [4000, 5200, 4800]
legend: true
beginAtZero: true
```

### Плагин **Charts View** (caronchen) — Ant Design Charts
Блок: ` ```chartsview `
Формат: массив объектов + `options`

```yaml
type: column
data:
  - month: Янв
    value: 4000
  - month: Фев
    value: 5200
options:
  xField: month
  yField: value
```

## 📚 Примеры по типам (Charts / phibr0)

### 📊 Bar / Column (Столбчатая)
```chart
type: bar
labels: [Продукты, Доставка, Хозтовары]
series:
  - title: План
    data: [15000, 3000, 2000]
  - title: Факт
    data: [18200, 4100, 2700]
legend: true
beginAtZero: true
```

### 📈 Line (Линейная)
```chart
type: line
labels: [Пн, Вт, Ср, Чт, Пт]
series:
  - title: Траты
    data: [1200, 980, 1500, 800, 2100]
tension: 0.3
fill: false
```

### 🥧 Pie (Круговая)
```chart
type: pie
labels: [Еда, Транспорт, Жильё, Развлечения]
series:
  - title: Траты
    data: [12000, 3500, 18000, 4200]
legend: true
```

### 🍩 Doughnut (Кольцевая)
```chart
type: doughnut
labels: [Белки, Жиры, Углеводы]
series:
  - title: БЖУ
    data: [120, 65, 210]
cutout: 50%
legend: true
```

### 🕸️ Radar (Пентаграмма)
```chart
type: radar
labels: [Сила, Ловкость, Интеллект, Харизма, Выносливость]
series:
  - title: Профиль
    data: [80, 65, 90, 45, 70]
fill: true
beginAtZero: true
max: 100
```

### 🎯 PolarArea (Полярная)
```chart
type: polarArea
labels: [Сон, Работа, Спорт, Хобби, Еда]
series:
  - title: Часы
    data: [7, 8, 2, 3, 4]
legend: true
```

### 🔹 Scatter (Точечная)
```chart
type: scatter
labels: []
series:
  - title: Данные
    data: [
        {x: 10, y: 20},
        {x: 15, y: 35},
        {x: 20, y: 25}
      ]
legend: true
```

### 🔵 Bubble (Пузырьковая)
```chart
type: bubble
labels: []
series:
  - title: Проекты
    data: [
        {x: 10, y: 20, r: 15},
        {x: 25, y: 10, r: 10}
      ]
legend: true
```

## ⚙️ Общие параметры (для любого типа)

```yaml
# Размеры
width: 100%
height: 400px

# Легенда и заголовок
legend: true
title: "Мой график"

# Оси
beginAtZero: true
scales:
  y:
    min: 0
    max: 100

# Стиль
fill: true
tension: 0.4
colors:
  - "#4a90e2"
  - "#e24a4a"

# Анимация (отключить для стабильности)
animation: false
```

## 💡 Интеграция с данными

### Из таблицы в заметке
```chart
type: bar
labels: [${table:Категория}]
series:
  - title: Сумма
    data: [${table:Сумма}]
```

### Из CSV-файла
```chart
type: line
csv: "путь/к/файлу.csv"
xField: date
yField: amount
```

### Через DataviewJS (динамически)
````js
```dataviewjs
const data = dv.pages("#кухня/траты")
  .filter(p => p.date)
  .map(p => ({x: p.date, y: p.sum}))

const chart = {
  type: "line",
  data: { labels: data.map(d => d.x), datasets: [{ data: data.map(d => d.y) }] }
}
console.log(`\`\`\`chart\n${JSON.stringify(chart)}\n\`\`\``)
```
````

## ⚠️ Ограничения и советы

1. Блок кода должен начинаться с ` ```chart ` или ` ```chartsview ` **без отступов**.
2. `labels` и `data` должны совпадать по длине.
3. Для `scatter`/`bubble` используй `labels: []` и передавай `{x, y, r}`.
4. Плагин не выполняет JS внутри блока — только статичные значения.
5. При изменении данных сохрани заметку или переключи режим просмотра.
6. Даты — в формате `YYYY-MM-DD` или `MM/YYYY`.
