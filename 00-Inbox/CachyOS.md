# CachyOS

CachyOS — это оптимизированный дистрибутив на базе Arch Linux, который ставит во главу угла производительность и безопасность. В отличие от стандартного Arch, он предлагает готовую к работе систему с предустановленными оптимизациями, которые обычно требуют ручной настройки.

## Что делает CachyOS особенным

Основная фишка дистрибутива — это кастомное ядро с набором патчей для улучшения отзывчивости системы. Здесь используются альтернативные планировщики процессов: BORE (стоит по умолчанию), EEVDF, TT и BMQ. Они меняют то, как система распределяет процессорное время между задачами, что особенно заметно при работе с интерактивными приложениями.

Пакеты в репозиториях скомпилированы с оптимизациями под современные процессоры (x86-64-v3/v4) и используют LTO (Link Time Optimization). Это даёт прирост производительности без каких-либо действий со стороны пользователя.

## Для кого этот дистрибутив

CachyOS подойдёт тем, кто хочет получить Arch-подобную систему без необходимости ручной настройки. Особенно он интересен геймерам — оптимизации ядра и планировщиков дают дополнительный FPS в играх, а установка драйверов и игровых утилит упрощена.

Также это хороший выбор для разработчиков, которым важна скорость компиляции и отзывчивость системы при работе с тяжёлыми проектами.

## Установка

Дистрибутив поставляется с графическим установщиком, который проводит пользователя через все этапы: разметка диска, выбор файловой системы, выбор окружения рабочего стола. Из коробки доступны KDE Plasma, GNOME, XFCE, а также тайловые оконные менеджеры вроде i3 и Hyprland.

Рекомендуется использовать файловую систему Btrfs с прозрачным сжатием zstd — это даёт экономию места на диске и ускоряет работу с файлами.

## После установки

Система обновляется через стандартный pacman или через утилиту cachyos-update. Для установки ПО из AUR используется yay. Отдельного внимания заслуживает cachyos-kernel-manager — утилита для управления ядрами, позволяющая легко переключаться между версиями или устанавливать альтернативные ядра (например, для Surface-устройств).

## Безопасность

В дистрибутиве по умолчанию включён firewall, доступны профили AppArmor, есть поддержка Secure Boot. Для тех, кто хочет максимальной защиты, доступно hardened ядро с дополнительными патчами безопасности.

## Итог

CachyOS — это Arch Linux для тех, кто ценит время. Дистрибутив берёт на себя всю рутину по оптимизации, оставляя пользователю доступ ко всем возможностям Arch: свежим пакетам, AUR и rolling-release модели обновлений.
