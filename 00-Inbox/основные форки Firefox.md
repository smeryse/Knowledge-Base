**Floorp** — японский форк с упором на кастомизацию. С версии 12 использует нативные вертикальные вкладки Firefox, плюс есть рабочие пространства, жесты мышью и заметки [[1]].

**Zen Browser** — открытый форк, вдохновлённый Arc. Вертикальные вкладки — это базовая часть дизайна, горизонтальные не планируются [[33]]. Есть сплит-вью и рабочие пространства [[30]].

**Waterfox** — с версии 6.6.0 добавил нативные вертикальные вкладки с древовидной структурой и автогруппировкой [[42]][[45]]. Также есть интеграция с расширением Tree Style Tab [[47]].

**LibreWolf** — приватный форк. Поддерживает нативные вертикальные вкладки Firefox 136+, включаются через `about:config`: `sidebar.verticalTabs = true` [[15]][[16]].

**Важно:** начиная с Firefox 136 вертикальные вкладки появились и в официальном браузере [[1]][[2]]. Если не хочешь менять браузер — можно включить их там, либо использовать расширения: **Sidebery** или **Tree Style Tab** [[49]][[54]].

Если нужны детали по настройке конкретного форка — уточни, распишу.
