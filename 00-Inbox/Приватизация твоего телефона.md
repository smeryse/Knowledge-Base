## 🔴 НИКОГДА не вставляй SIM-карту при покупке телефона

**Почему (из видео + факты):**
- Базовый процессор (Baseband Processor, BP) работает автономно от основного и передаёт в сеть данные о устройстве, вышках и геолокации **без твоего контроля**, даже если ты видишь «авиарежим» [[1]].
- Как только SIM-карта вставлена хотя бы раз — оператор получает полный профиль: устройство, перемещения, паттерны поведения.
- Номер телефона — это **арендуемый ресурс оператора**, а не твоя собственность.
- Протоколы **SS7 / Diameter** имеют уязвимости: злоумышленник с доступом к ним может выполнить команду `Anytime Interrogation` и получить твоё местоположение с точностью до здания [[25]].
- **SIM-свопинг**: через уязвимости или социнженерию злоумышленник может перевыпустить твою SIM и перехватить 2FA-коды от банков и аккаунтов.

**Правило**: хочешь приватности — работай только через Wi-Fi, без SIM. Это не паранойя, а осознанный выбор автора видео.

## ⚠️ Смена IMEI: юридический статус в РФ

**Коротко: не делай этого.**

- С 2026 года в РФ действует закон об **обязательной регистрации IMEI** при привязке SIM-карты [[15]][[18]].
- Изменение IMEI классифицируется как **административное или уголовное правонарушение** (ст. 13.3 КоАП РФ, а при отягчающих — УК РФ).
- Технически методы существуют (через инженерное меню, патчинг прошивки, EDL-режим), но:
  - Требуют root / разблокированного загрузчика;
  - Работают не на всех чипсетах;
  - Оставляют логи в системе и на стороне оператора;
  - **Не дают приватности**: отслеживание идёт не только по IMEI, но и по MAC, рекламным идентификаторам, поведению в приложениях.

**Вывод**: смена IMEI — незаконна, неэффективна для приватности и создаёт больше рисков, чем пользы.

## 📡 Как отслеживают человека через сотовые сети (техническая сторона)

**На уровне оператора / спецслужб:**
1. **Triangulation по вышкам**: устройство постоянно обменивается сигналами с ближайшими БС; по задержке и мощности сигнала вычисляется местоположение.
2. **SS7 / Diameter-атаки**: команда `Anytime Interrogation` или `Provide Subscriber Location` возвращает координаты абонента [[24]][[25]].
3. **VoLTE / VoWiFi туннели**: даже без SIM устройство может установить зашифрованный канал к шлюзу оператора через интернет, передавая метаданные.
4. **IMSI-catchers (Stingray)**: имитируют базовую станцию, заставляют телефон подключиться и раскрывать идентификаторы.

**На уровне приложений / ОС:**
- Рекламный идентификатор (AAID / GAID) + GPS + сканирование Wi-Fi/Bluetooth-точек = детальный профиль перемещений.
- Данные агрегируются брокерами (например, платформы типа Fog Data) и продаются легально.

**Защита**: отключение радиомодулей на аппаратном уровне, использование устройств с физическими «килл-свитчами» (PinePhone, Librem 5), работа только через доверенный Wi-Fi с контролем трафика.

## 🐧 Kali NetHunter на твой телефон

**Сначала определи: подходит ли твоё устройство.**

Официально поддерживаемые устройства: [[7]][[9]]
- Google Pixel 3/4/4a/5/6 серии
- OnePlus 7/7T/8, Poco F1 / X3 Pro
- Старые Nexus (5, 6P, 7) — уже устарели

**Варианты установки:**

### 🔹 Rootless (без рута, самый безопасный для начала)
1. Установи **Termux** (с F-Droid, не из Play Store).
2. В Termux выполни:
   ```bash
   pkg install wget -y
   wget -O install-nethunter-termux https://offs.ec/2MceZWr
   chmod +x install-nethunter-termux
   ./install-nethunter-termux
   ```
3. Установи **NetHunter KeX** для графического интерфейса.
4. Запускай: `nethunter kex &` → подключайся через VNC-клиент.

**Плюсы**: не требует рута, не ломает телефон.  
**Минусы**: нет доступа к монитор-режиму для Wi-Fi, ограниченный контроль над железом.

### 🔹 Full (с рутом и кастомным ядром)
Требует:
- Разблокировки загрузчика
- Установки TWRP
- Root через Magisk
- Прошивки кастомного ядра с поддержкой инъекции пакетов и монитор-режима [[2]][[9]]

**Плюсы**: полный контроль, поддержка внешних Wi-Fi-адаптеров для аудита.  
**Минусы**: риск «кирпича», потеря гарантии, сложность, не для ежедневного использования.

### 🔹 Для твоего случая (студент, информационная безопасность, Ubuntu, OSINT)
- Если телефон **не в списке поддерживаемых** — начни с **rootless-версии** в Termux: достаточно для изучения сетевого трафика, базового пентеста, работы с инструментами типа `nmap`, `wireshark`, `sqlmap`.
- Для серьёзного аудита: купи **поддерживаемый б/у Pixel 4a** или используй **внешний адаптер с ноутбука** на Kali.
- Долгосрочно: рассмотри **PinePhone Pro** или **Librem 5** — аппаратные переключатели, чистый Linux, контроль над радиомодулями [[10]].
