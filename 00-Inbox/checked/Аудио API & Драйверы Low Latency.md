## **Windows аудио API**
```text
[Аудиоинтерфейс]
       │
       ▼
     ASIO  ───> Reaper (минимальная задержка, многоканально)
       │
       ▼
WASAPI Exclusive ───> Reaper (низкая задержка, прямой доступ)
       │
       ▼
WASAPI Shared / DirectSound ───> Windows микшер ───> Reaper
       ↑
   Задержка выше, совместный вывод, дополнительный оверхед
```
1. **ASIO**
    - Драйвер низкой задержки для DAW.
    - Минус: нужен отдельный ASIO-драйвер для интерфейса.
    - Плюс: прямой доступ к железу, минимальная латентность, многоканальный ввод/вывод.
2. **WASAPI**
    - Современная Windows подсистема.
    - **Exclusive mode** → Reaper «монополизирует» устройство → минимальная задержка (как ASIO).
    - **Shared mode** → аудио делится с другими приложениями → задержка выше, обычный звук Windows.
    - Когда работает какой режим:
        - Exclusive → DAW, студийная запись.
        - Shared → мультимедиа, проигрывание музыки, видео.
3. **DirectSound**
    - Старый API, через него Reaper идёт через Windows микшер.
    - **Оверхед** → лишние промежуточные шаги: микширование, буферизация, копирование данных → высокая задержка.
    - Практически не используют для записи/живого исполнения.

## **Linux аудио API**
```
[Аудиоинтерфейс]
       │
       ▼
     ALSA ───> Reaper (прямой доступ, низкая латентность, простой ввод/вывод)
       │
       ▼
     JACK ───> Reaper + другие приложения (минимальная задержка, синхронизация, маршрутизация)
       │
       ▼
 PulseAudio / PipeWire ───> Reaper (высокая задержка, для мультимедиа)

```

1. **ALSA**
    - Драйвер ядра, прямой доступ к устройству.
    - Латентность ограничена настройками буферов, простой ввод/вывод.
2. **JACK**
    - Система маршрутизации аудио для профессиональной работы.
    - Низкая латентность, синхронизация между приложениями, многоканальные соединения.
3. **PulseAudio / PipeWire**
    - Для обычного десктопа, стриминга и мультимедиа.
    - Высокая задержка, не для live-записи.
    - PipeWire может работать как PulseAudio или JACK (универсально).

### **Ключевые соответствия**

| Windows                     | Linux       | Особенности                              |
| --------------------------- | ----------- | ---------------------------------------- |
| ASIO                        | JACK        | Минимальная задержка, многоканально, DAW |
| WASAPI exclusive            | ALSA direct | Низкая задержка, простой ввод/вывод      |
| WASAPI shared / DirectSound | PulseAudio  | Домашний звук, высокая задержка          |
