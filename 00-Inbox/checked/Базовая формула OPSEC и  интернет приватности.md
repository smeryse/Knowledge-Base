# 1. Базовая “формула OPSEC”:

Используется людьми, которые хотят исчезнуть из цифрового пространства.

### Формула 1. “MXP” — Minimize, eXchange, Protect

M — Minimize (минимизировать):

Убрать всё лишнее: старые аккаунты, почты, номера, привязки, сканы документов, фото.

X — eXchange (заменить):

Номер → виртуальный

Почта → отдельная под каждый сервис

Логины → уникальные

Пароли → менеджер

P — Protect (защитить):

2FA

Аппаратная ключи (если нужно)

Контейнеры (Sandbox)

Изолированные профили браузера

# 2. Методика “6 Circles of Identity”

Шестикруговая модель, чтобы понять, где именно остался след:

1. Legal ID — паспорт, номер телефона, ИНН
    
2. Financial ID — банковские карты
    
3. Core ID — основная почта, основные мессенджеры
    
4. Usage ID — сервисы, соцсети
    
5. Metadata ID — куки, профили браузера, отпечатки
    
6. Shadow ID — базы утечек, форумы, зеркала
    

Ты проходишь по кругам, чистишь и изолируешь каждый слой.

# 3. Привычки, которые обнуляют цифровой след

Это прям «формулы поведения»:

### Формула “1 аккаунт = 1 почта = 1 пароль”

Никаких пересечений.

Даже если сервис утечёт — другие не пострадают.

### Формула “KyS” — Keep Yourself Separate

Телефон под друзей ≠ телефон под банки ≠ телефон под сервисы.

### Формула “NF-PM” — No Forwarding, Private Mail

Не форвардить почту между ящиками.

Каждый ящик — изолированный.

# 7. Полный алгоритм очистки следа

Твоя стратегия сейчас правильная, но вот более профессиональный порядок:

### Шаг 1. Inventory (Создать полный список следов)

— Все почты

— Все номера

— Все аккаунты

— Все утечки (чекаешь через OSINT)

### Шаг 2. Disabling (отключение привязок)

— удаляешь номера

— удаляешь бэкап-почты

— меняешь телефоны на виртуальные

### Шаг 3. Disassembly (разборка личности)

— удаление/запрос удаления аккаунтов

— удаление персональных данных согласно 152‑ФЗ

— удаление старых соцсетей

### Шаг 4. Recreation (пересборка)

Создаёшь новую структуру:

— новые почты

— новые логины

— новые аккаунты

полностью изолированные

### Шаг 5. Protection (защита)

— Встроенные контейнеры

— VPN

— Randomized User-Agent

— Private DNS (AdGuard/Public Resolver)
