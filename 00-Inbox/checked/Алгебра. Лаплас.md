# Кластеризация графов
Это процесс разбиения графов на группы так, чтобы:
- Внутри группы - вершины сильно связаны
- Между группами - слабо связаны

# Спектральная кластеризация
1. Формируем

# Матрица Лапласа
$$L=D-A$$
$D - \text{диагональная матрица степеней}$
$A - \text{матрица смежности}$
# Собственный вектора матрицы Лапласа
$$|L-\lambda I|v  = 0$$
$v - \text{собственный вектор матрицы Лапласа}, \ \lambda - \text{собственное значение матрицы Лапласа}$

# Собственные значения матрицы
$$det|A-\lambda I|=0$$
$\text{где } \lambda \text{ - собственное значение матрицы}$

# Фундаментальные свойства матриц
Направление, растяжение \ сжатие
