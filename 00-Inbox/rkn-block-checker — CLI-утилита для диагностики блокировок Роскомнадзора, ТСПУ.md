**Суть:** проверяет доступность сайтов слой за слоем (DNS → TCP → TLS → HTTP) и определяет *тип* блокировки, а не просто констатирует факт недоступности.

**Что детектирует:**
- **DNS poisoning** — подмена ответов в системном резолвере
- **TCP reset** — разрыв соединения на уровне IP
- **TLS DPI (SNI)** — блокировка по домену в TLS-рукопожатии (основной метод ТСПУ)
- **HTTP stub** — подмена страницы провайдером (заглушка с 200/451)

**Базовое использование:**
```bash
rkn-check                          # полная проверка по встроенным спискам
rkn-check --url example.com       # проверка конкретного сайта
rkn-check --json | jq ...         | машинный вывод для скриптов
rkn-check --no-self-info          # без запроса вашего публичного IP
```

**Особенности:**
- Встроенные списки: ~21 «белый» сайт + ~15 заблокированных
- Сравнение системного DNS с Cloudflare DoH для детекта отравления
- Вердикты с уровнем уверенности (HIGH/MEDIUM/LOW)
- Нет телеметрии, минимальный след в логах (стандартный User-Agent)
- Поддержка Docker, кастомных списков, JSON-вывода

**Установка:** `pip install rkn-block-checker`

У тебя уже стоит в `/data/06-Software/Installed/rkn-block-checker` — можно запускать напрямую или через `python -m rkn_checker`.
