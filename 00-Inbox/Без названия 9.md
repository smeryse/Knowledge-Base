## Подготовка

- [ ] Скачать DeltaChat и создать Яндекс.Почту (или использовать существующую)
- [ ] Перейти в [Настройка почтовых клиентов](https://yandex.ru/support/mail/mail-clients.html)
- [ ] Включить опцию «С сервера imap.yandex.ru по протоколу IMAP»
- [ ] Убедиться, что включён «Пароли приложений и OAuth-токены»
- [ ] Сохранить почтовый адрес в надёжном месте
- [ ] Перейти в [Пароли приложений](https://id.yandex.ru/security/app-passwords)
- [ ] Выбрать категорию «Почта IMAP, POP3, SMTP»
- [ ] Придумать имя для пароля (не «DeltaChat» — может светиться в логах) → «Далее»
- [ ] Скопировать и сохранить сгенерированный пароль приложения

> [!IMPORTANT]
> Используйте именно пароль приложения, а не основной пароль от Яндекс.Почты.

---

## Создание профиля в DeltaChat

- [ ] При первом запуске: «Создать новый профиль»
- [ ] При добавлении на новом устройстве: «У меня уже есть профиль»

### Имя и аватар
- [ ] Ввести отображаемое имя
- [ ] (Опционально) Добавить аватар
- [ ] Нажать «Использовать другой сервер»

### Тип подключения
- [ ] Выбрать «Использовать электронную почту как реле»

### Учётные данные
| Поле | Значение |
|------|----------|
| Адрес почты | `ваш_логин@yandex.ru` |
| Пароль | _пароль приложения_ (не основной!) |

## Завершение

- [ ] Дождаться подтверждения подключения
- [ ] Для добавления контакта: иконка QR-кода в правом верхнем углу → отсканировать код собеседника

> [!TIP]
> Если подключение не устанавливается: проверьте, включён ли IMAP в настройках Яндекс.Почты, и что пароль приложения скопирован без лишних пробелов.

---

## Параметры сервера (для ручного ввода)

```
IMAP-сервер: imap.yandex.ru, порт 993, SSL/TLS
SMTP-сервер: smtp.yandex.ru, порт 465, SSL/TLS
```

> [!NOTE]
> DeltaChat обычно подставляет эти данные автоматически при выборе Яндекс.Почты. Ручной ввод нужен редко.
