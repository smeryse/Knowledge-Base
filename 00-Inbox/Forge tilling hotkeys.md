# Forge (GNOME) — шпаргалка по тайлингу 🧩

Эта шпаргалка основана на **твоих текущих биндингах Forge** в Ubuntu (GNOME).  
`Super` = клавиша **Win**.

---

## 🧭 Навигация между окнами

| Действие                | Комбинация      |
| ----------------------- | --------------- |
| Фокус вниз              | `Super + j`     |
| Фокус влево             | `Super + h`     |
| Фокус вправо            | `Super + l`     |
| Фокус вверх             | `Super + k`     |
| Последнее активное окно | `Super + Enter` |

> Мнемоника: **hjkl** как в *vim*.

---

## 🚚 Перемещение окон

| Действие           | Комбинация          |
| ------------------ | ------------------- |
| Переместить вниз   | `Shift + Super + j` |
| Переместить влево  | `Shift + Super + h` |
| Переместить вправо | `Shift + Super + l` |
| Переместить вверх  | `Shift + Super + k` |

---

## 📐 Изменение размеров окон (Resize)

### Левая граница
| Действие | Комбинация |
|----------|------------|
| Увеличить (расширить влево) | `Ctrl + Super + h` |
| Уменьшить (сузить слева) | `Shift + Ctrl + Super + h` |

### Нижняя граница
| Действие | Комбинация |
|----------|------------|
| Увеличить (расширить вниз) | `Ctrl + Super + j` |
| Уменьшить (сузить снизу) | `Shift + Ctrl + Super + j` |

### Верхняя граница
| Действие | Комбинация |
|----------|------------|
| Увеличить (расширить вверх) | `Ctrl + Super + k` |
| Уменьшить (сузить сверху) | `Shift + Ctrl + Super + k` |

### Правая граница
| Действие | Комбинация |
|----------|------------|
| Увеличить (расширить вправо) | `Ctrl + Super + l` |
| Уменьшить (сузить справа) | `Shift + Ctrl + Super + l` |

> 💡 **Логика:** `Ctrl + Super` — увеличить, `+ Shift` — уменьшить.

---

## 📏 Gaps (отступы между окнами)

| Действие | Комбинация |
|----------|------------|
| Уменьшить gaps | `Ctrl + Super + -` |
| Увеличить gaps | `Ctrl + Super + +` |

---

## 🧱 Контейнеры и лейауты

| Действие | Комбинация |
|----------|------------|
| Split горизонтально | `Super + g` |
| Stacked layout | `Shift + Super + s` |
| Tabbed layout | `Shift + Super + t` |
| Toggle tab decorations | `Ctrl + Alt + y` |

---

## 🪟 Float режим

| Действие     | Комбинация          |
| ------------ | ------------------- |
| Toggle float | `Super + c`         |
| Always float | `Shift + Super + c` |

| Toggle tiling (глобально) | `Super + w` |

## 🎨 Внешний вид

| Действие | Комбинация |

---

## ⚙️ Forge / Preferences

| Действие | Комбинация |
|----------|------------|
| Открыть настройки Forge | `Super + .` |

---

## 🧪 Рекомендуемый workflow (для привыкания)

1. **Открываешь окна** обычным способом
2. `Super + g` — делишь экран (сплит)
3. `Super + hjkl` — навигация между окнами
4. `Shift + Super + hjkl` — двигаешь окна местами
5. `Ctrl + Super + hjkl` — меняешь размер границ
6. `Ctrl + Super + +/-` — настраиваешь gaps
7. `Super + c` — если окно должно «плавать»

---

## 🎯 Быстрая память

| Модификатор | Действие |
|-------------|----------|
| `Super` | Фокус (куда смотреть) |
| `Shift + Super` | Перемещение (куда тащить) |
| `Ctrl + Super` | Размер (тянуть край) |
| `+ Shift` к размеру | Уменьшать (обратное) |
| `hjkl` | ← ↓ ↑ → (как в Vim) |
