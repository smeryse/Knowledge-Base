# Insular/Island: изоляция «стукачей» на Android

> **Цель статьи**: инструкция по настройке песочницы Insular (ранее Island) для изоляции приложений, которые могут сообщать о наличии VPN-соединения.

## 🔑 Ключевая идея
Приложения из «жизненно важного списка» могут детектировать VPN и отказываться работать. Решение — поместить такие приложения в изолированный рабочий профиль (песочницу), где:
- весь трафик идёт через единый сетевой интерфейс (можно настроить обход VPN для всей песочницы);
- приложения не имеют доступа к контактам, фото, файлам основной системы;
- их можно массово «замораживать» одним действием.

## 📦 Установка

### 1. Установить F-Droid
- Скачать с [f-droid.org](https://f-droid.org)
- Это репозиторий свободного ПО без рекламы и платных функций

### 2. Установить Insular через F-Droid
- Открыть F-Droid → поиск → `Insular`
- Установить приложение

> **Примечание**: ранее приложение называлось *Island*, сейчас — *Insular*.

---

## 🚀 Настройка и использование

### Базовый принцип
- **MAINLAND** — основная система (ваши обычные приложения)
- **ISLAND** — изолированная песочница («цифровой ГУЛАГ» для подозрительных приложений)

### Пошаговая изоляция приложения (на примере Яндекс.Навигатора):
1. Открыть Insular → вкладка `MAINLAND`
2. Найти нужное приложение в списке
3. Выделить его → нажать `Island` (в нижнем меню)
4. Дождаться установки копии в песочницу
5. Перейти во вкладку `ISLAND` — убедиться, что приложение появилось там
6. **Удалить оригинал из основной системы**:
   - Вернуться в `MAINLAND`
   - Найти приложение → три точки → `Uninstall`
7. (Опционально) Создать ярлык на рабочий стол:
   - В `ISLAND` выбрать приложение → шестерёнка → `Create Unfreeze & Launch shortcut`

> ⚠️ Ярлык изолированного приложения будет помечен специальной иконкой.

## ✅ Преимущества
- Подозрительные приложения вынесены из основного контура
- Единая сетевая политика для всей песочницы (удобно для настройки VPN)
- Ограниченный доступ к личным данным (контакты, фото, файлы)
- Многие приложения перестают жаловаться на VPN после изоляции
- Не требует root-прав

## ⚠️ Ограничения
- Не полноценная виртуальная машина — изоляция не абсолютная
- Не все приложения корректно работают в рабочем профиле
- Уведомления, обмен файлами, интеграция с системой могут работать с ограничениями
- Требуется ручная настройка для некоторых приложений
- При ужесточении контроля могут потребоваться дополнительные меры

## 💬 Критика из комментариев
- Некоторые пользователи сообщают, что приложения (Мегафон, ВБ, банки) всё равно детектируют VPN даже в песочнице
- Возможная причина: приложения проверяют системные уведомления или DNS-настройки, а не только сетевой интерфейс
- Альтернативное решение: отключить системное уведомление «VPN подключён» через ADB + проверить настройки split-tunneling в самом VPN-клиенте
