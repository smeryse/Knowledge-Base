# Использование Crush CLI в Ubuntu Linux (установлено через npm)

## Установка

Для установки Crush CLI через npm выполните команду:

```bash
npm install -g @crush/cli
```

Параметр `-g` означает глобальную установку, чтобы команда была доступна из любого места.

## Проверка установки

Проверьте, что Crush CLI установлен и доступен:

```bash
crush --version
```

Если версия отображается, значит установка прошла успешно.

## Основные команды Crush CLI

- `crush init` — инициализация нового проекта Crush.
- `crush build` — сборка проекта.
- `crush serve` — запуск локального сервера для разработки.
- `crush deploy` — деплой проекта (если настроено).
- `crush help` — вывод справки по командам.

## Использование Crush CLI вместе с LM Studio

LM Studio — это среда разработки и управления проектами, которая может интегрироваться с Crush CLI для упрощения рабочих процессов.

### Шаги интеграции

1. **Убедитесь, что Crush CLI установлен и работает**  
   Проверьте командой:

   ```bash
   crush --version
   ```

2. **Откройте проект в LM Studio**  
   Запустите LM Studio и откройте папку с вашим проектом, созданным через Crush CLI (`my-crush-project` или другой).

3. **Настройка задач (tasks) в LM Studio**  
   В LM Studio можно настроить задачи для запуска команд Crush CLI напрямую из интерфейса:

   - Создайте новую задачу для сборки проекта:

     ```bash
     crush build
     ```

   - Задача для запуска локального сервера разработки:

     ```bash
     crush serve
     ```

   - Другие задачи по необходимости (`crush deploy`, `crush init` и т.д.)

4. **Автоматизация и отладка**  
   Используйте возможности LM Studio для отладки и мониторинга процессов, запущенных через Crush CLI.

5. **Использование терминала LM Studio**  
   Встроенный терминал LM Studio позволяет запускать любые команды Crush CLI без необходимости переключаться в отдельное окно терминала.

### Преимущества связки

- Удобное управление проектом через графический интерфейс LM Studio.
- Быстрый запуск и мониторинг команд Crush CLI.
- Возможность интеграции с другими инструментами и плагинами LM Studio.

## Пример использования

1. Создайте новую папку и перейдите в неё:

```bash
mkdir my-crush-project
cd my-crush-project
```

2. Инициализируйте проект:

```bash
crush init
```

3. Запустите локальный сервер для разработки:

```bash
crush serve
```

4. Откройте в браузере адрес, который выведет команда (обычно http://localhost:3000).

## Полезные советы

- Для обновления Crush CLI используйте:

```bash
npm update -g @crush/cli
```

- Если команда `crush` не найдена, убедитесь, что путь к npm глобальным пакетам добавлен в переменную окружения PATH.

- Ознакомьтесь с документацией Crush CLI для расширенных возможностей и конфигураций.
