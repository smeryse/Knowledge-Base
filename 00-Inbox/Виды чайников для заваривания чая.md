## По конструкции и методу заваривания

### Классический заварник
- **Материалы**: фарфор, стекло, керамика
- **Метод**: европейский (настой 3-5 мин)
- **Плюсы**: универсальность, простота
- **Минусы**: риск перезаваривания

### Френч-пресс
- **Конструкция**: стеклянная колба + поршень-фильтр
- **Плюсы**: 
  - не нужны дополнительные ситечки
  - хорош для крупных листьев и трав
- **Минусы**: чай продолжает контактировать с водой → возможна горечь

### Гайвань
- **Объём**: 70-150 мл
- **Материал**: фарфор/керамика
- **Метод**: короткие проливы
- **Для каких чаёв**: зелёные, белые, светлые улуны
- **Плюсы**: максимальный контроль, не искажает вкус
- **Идеально**: для дегустации

### Исинский чайник
- **Материал**: пурпурная глина
- **Особенности**: 
  - глина «дышит», впитывает чайные масла
  - усиливает вкус со временем
- **Правило**: один чайник → один тип чая
- **Для каких чаёв**: пуэры, тёмные улуны
- **Минусы**: требует ухода, дорогой

### 🔥 Чайник с отключаемым ситечком (типод)
- **Принцип**: кнопка → ситечко уходит вниз
- **Плюсы**: исключает перезаваривание
- **Идеально**: для офиса

### Чахай (сливник)
- **Назначение**: не для заваривания, а для переливания готового настоя
- **Функция**: выравнивает крепость, предотвращает перезаваривание

## Влияние материала на вкус

| Материал | Характеристики | Для каких чаёв |
|----------|----------------|----------------|
| **Фарфор/фаянс** | Нейтральный, не впитывает запахи | Универсально |
| **Стекло** | Инертное, видно цвет настоя, не держит тепло | Любой чай |
| **Глина (исинская)** | Меняет текстуру, делает мягче и плотнее | Пуэры, улуны |
| **Керамика глазурованная** | Нейтральность + удержание тепла | Универсально |
| **Чугун** | Отлично держит тепло, тяжёлый | Обычно с керамическим вкладышем |

## Как выбрать

- **Изучать чай, ловить нюансы** → фарфоровая гайвань
- **Пуэры/тёмные улуны, плотный настой** → исинский чайник
- **Просто и удобно (дом/офис)** → классический заварник с ситечком или типод
- **Травяные сборы, крупнолистовой чёрный** → френч-пресс
- **Чай в дорогу** → термокружка со съёмным ситечком
