## 📦 Активные плагины

| Плагин                  | Что делает                                      | Ссылка                                                   |
| ----------------------- | ----------------------------------------------- | -------------------------------------------------------- |
| **Dataview**            | Таблицы, списки, запросы к базы знаний          | `obsidian://show-plugin?id=dataview`                     |
| **Obsidian Charts**     | Графики и диаграммы                             | `obsidian://show-plugin?id=obsidian-charts`              |
| **Git**                 | Версионирование заметок через Git               | `obsidian://show-plugin?id=obsidian-git`                 |
| **Calendar**            | Календарь для ежедневных заметок                | `obsidian://show-plugin?id=calendar`                     |
| **Tasks**               | Управление задачами (todo-листы)                | `obsidian://show-plugin?id=obsidian-tasks-plugin`        |
| **Pomodoro Timer**      | Таймер Pomodoro для фокусировки                 | `obsidian://show-plugin?id=pomodoro-timer`               |
| **Terminal**            | Терминал внутри Obsidian                        | `obsidian://show-plugin?id=terminal`                     |
