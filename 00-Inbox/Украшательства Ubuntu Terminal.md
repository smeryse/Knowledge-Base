# Украшательства Ubuntu Terminal

## cool-retro-term  
Эмулятор терминала с ретро-стилистикой.

**Запуск и модификаторы:**  
- `cool-retro-term` — запуск с настройками по умолчанию.  
- `cool-retro-term --help` — показать все доступные опции.  
- `cool-retro-term --profile <имя_профиля>` — запуск с определённым профилем (темой).  
- `cool-retro-term --fullscreen` — запуск в полноэкранном режиме.  
- `cool-retro-term --geometry WxH+X+Y` — задать размер и позицию окна (например, `80x24+100+100`).  

**Настройка:**  
- В меню настроек можно менять темы, эффекты (помехи, свечения), шрифты.  
- Профили сохраняются и загружаются для быстрого переключения.

## cmatrix  
Анимация "Матрица" в терминале.

**Основные опции:**  
- `cmatrix` — запуск с настройками по умолчанию.  
- `cmatrix -s` — медленное движение символов (slow).  
- `cmatrix -b` — яркий режим (bold characters).  
- `cmatrix -u <число>` — скорость обновления в миллисекундах (например, `-u 50`).  
- `cmatrix -C <цвет>` — цвет текста (например, green, red, blue).  
- `cmatrix -x` — использовать символы ASCII вместо юникода.  
- `cmatrix -V` — показать версию.  
- `cmatrix -h` — помощь.

**Пример:**  
```bash
cmatrix -b -u 30 -C red
```

## cava  
Консольный аудиовизуализатор.

**Запуск и опции:**  
- `cava` — запуск с конфигурацией по умолчанию.  
- `cava -p <путь_к_конфигу>` — использовать альтернативный конфигурационный файл.  
- `cava -h` — помощь.

**Конфигурация:**  
- Файл конфигурации обычно находится в `~/.config/cava/config`.  
- В нем можно настроить:  
  - Источник звука (alsa, pulse, fifo и др.)  
  - Цвета и символы визуализации  
  - Число полос и их ширину  
  - Чувствительность и масштабирование

**Пример запуска с кастомным конфигом:**  
```bash
cava -p ~/.config/cava/myconfig
```

## yamusic-tui  
Текстовый интерфейс для Яндекс.Музыки.

**Запуск и опции:**  
- `yamusic-tui` — запуск приложения.  
- `yamusic-tui --help` — показать доступные команды и опции.

**Управление в приложении:**  
- Стрелки — навигация.  
- Enter — выбрать/воспроизвести.  
- Пробел — пауза/воспроизведение.  
- q — выход из приложения.  
- / — поиск по музыке.  
- l — добавить в плейлист.

**Авторизация:**  
- При первом запуске потребуется авторизоваться через Яндекс аккаунт (следуй инструкциям в терминале).
