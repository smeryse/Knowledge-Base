# Контроль данных

## Товары без цены

```dataview
TABLE barcode as "Штрихкод", category as "Категория", brand as "Бренд"
FROM "LifeOS/Кухня/Products"
WHERE type = "product" AND (!price OR price = "")
SORT file.name ASC
```
