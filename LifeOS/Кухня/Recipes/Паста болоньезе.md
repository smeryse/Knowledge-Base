# Паста болоньезе

## Ингредиенты

| Продукт        | Кол-во | Ед. | Примечание |
| --------------- | ------ | --- | ---------- |
| Фарш            | 400    | г   | говяжий или смешанный |
| Лук             | 1      | шт  | средний |
| Морковь         | 1      | шт  | небольшая |
| Томаты          | 3      | шт  | свежие |
| Томатная паста  | 2      | ст.л | |
| Макароны        | 100    | г   | на 1 порцию |
| Подсолнечное масло | 1   | ст.л | |
| Чеснок          | 2      | зубч | по желанию |
| Соль            | 1      | ч.л | по вкусу |
| Чёрный перец    | 0.5    | ч.л | |
| Вода            | 150    | мл  | для соуса |

Для автосписания указывай числовое количество и единицы `г`, `кг`, `мл`, `л` или `шт`.

Все количества в рецепте указывай на 1 порцию. При готовке скрипт сам умножит их на нужное число порций.

## Шаги

1. Мелко нарезать лук и морковь.
2. Разогреть сковороду с подсолнечным маслом.
3. Обжарить лук 3–4 минуты, затем добавить морковь и готовить ещё 3 минуты.
4. Добавить фарш и жарить, разбивая комки, до лёгкой поджарки.
5. Натереть или мелко нарезать томаты.
6. Добавить томаты и томатную пасту к фаршу.
7. Влить воду, добавить соль и перец.
8. Тушить на слабом огне 20–30 минут.
9. Отдельно сварить макароны до состояния al dente.
10. Смешать макароны с соусом перед подачей.

## Заметки

> Если соус становится слишком густым — добавить немного воды.
> 
> Лучше не делать соус слишком томатным: в болоньезе основной вкус должен быть мясным.
