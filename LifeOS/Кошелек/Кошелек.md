## Сводка за текущий месяц

```dataview
TABLE WITHOUT ID
  link(file.name, month_key) as "Месяц",
  total_plan as "План",
  total_fact as "Факт",
  (total_plan - total_fact) as "Остаток"
FROM "LifeOS/Кошелек/Budgets"
WHERE month_key = "2026-05"
```

## Счета и балансы

```dataview
TABLE WITHOUT ID
  link(file.name, title) as "Счёт",
  account_type as "Тип",
  balance as "Баланс",
  currency as "Валюта"
FROM "LifeOS/Кошелек/Accounts"
SORT balance DESC
```

---

## Цели накоплений

```dataview
TABLE WITHOUT ID
  link(file.name, title) as "Цель",
  target_amount as "Цель",
  saved_amount as "Накоплено",
  round((saved_amount / target_amount) * 100) + "%" as "Прогресс",
  deadline as "Дедлайн"
FROM "LifeOS/Кошелек/Savings"
SORT deadline ASC
```

---

## Ближайшие регулярные платежи

```dataview
TABLE WITHOUT ID
  link(file.name, title) as "Платёж",
  amount as "Сумма",
  next_due as "Следующая дата",
  period as "Период"
FROM "LifeOS/Кошелек/Recurring"
SORT next_due ASC
LIMIT 5
```
