# Покупки

## Активные покупки

```dataview
TABLE product as "Товар", target_qty as "Нужно", unit as "Ед.", reason as "Зачем"
FROM "Projects/Кухня/Shopping List"
WHERE type = "shopping-item" AND status != "done" AND status != "cancelled"
SORT file.mtime DESC
```
