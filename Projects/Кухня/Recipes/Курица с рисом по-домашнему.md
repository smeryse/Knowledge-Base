# Курица с рисом по-домашнему

## Ингредиенты

| Продукт | Кол-во | Ед. | Примечание |
| ------- | ------ | --- | ---------- |
| Рис круглозерый | 80 | г | Можно заменить на длиннозерный |
| Куриное бедро или филе | 120 | г | Без кости и кожи |
| Морковь | 50 | г | Натереть на крупной тёрке |
| Лук репчатый | 30 | г | Нарезать кубиками |
| Масло растительное | 10 | мл | 1 ст. ложка |
| Вода | 200 | мл | Горячая |
| Соль | 0.5 | ч. л. | |
| Перец чёрный молотый | | по вкусу | |
| Куркума или паприка | 0.25 | ч. л. | Для цвета, по желанию |
| Чеснок | 1 | зубчик | Мелко порубить, по желанию |

## Шаги

1. Курицу нарезать небольшими кусочками (2–3 см). Морковь натереть на крупной тёрке, лук нарезать мелкими кубиками.
2. Разогреть масло в кастрюле или сковороде с высокими бортиками на среднем огне. Обжарить курицу 3–4 минуты, пока не появится румяная корочка.
3. Добавить лук и морковь. Обжаривать ещё 3 минуты, помешивая.
4. Добавить рис, перемешать и обжаривать 1 минуту — рис должен пропитаться маслом.
5. Залить горячей водой так, чтобы она накрывала рис примерно на 1 см (около 200 мл на порцию). Посолить, поперчить, добавить специи.
6. Накрыть крышкой, уменьшить огонь до минимума. Тушить 20–25 минут, **не открывая крышку**.
7. Выключить огонь. Если добавляете чеснок — мелко порубите и вмешайте сейчас. Дайте блюду постоять под крышкой ещё 5 минут.
8. Перемешать вилкой и разложить по контейнерам или подавать сразу.

## Заметки

> Идеальное блюдо для массовой готовки: в одной кастрюле на 3–4 л получается 6–10 порций. Время готовки той же — просто увеличьте воду пропорционально рису.
>
> Хорошо переносит разогрев в микроволновке и хранение в холодильнике 2–3 дня.
>
> **Примерная себестоимость порции:** ~70 ₽ (рис + курица + овощи). С упаковкой и специями — ~85–90 ₽. Отличная база для `[[Projects/СтудПит/СтудПит|СтудПит]]`.
