# Шпаргалка по категориям и единицам

Эта заметка нужна, чтобы при добавлении товаров не придумывать каждый раз категории, тип магазина и единицы измерения.

---

## Категории продуктов

Используй в первую очередь именно эти значения.

| Категория | Когда использовать | Примеры |
| --- | --- | --- |
| `молочка` | Молочные и кисломолочные продукты | молоко, кефир, йогурт, творог, сметана |
| `яйца` | Яйца и яичные наборы | куриные яйца, перепелиные яйца |
| `сладости` | Всё сладкое и десертное | вафли, шоколад, печенье, конфеты |
| `напитки` | Напитки любого типа | вода, сок, кола, энергетик |
| `крупы` | Крупы, макароны, мука, сухие основы | рис, гречка, овсянка, макароны, мука |
| `мясо` | Мясо, птица, колбасы, мясные полуфабрикаты | курица, фарш, сосиски, ветчина |
| `заморозка` | Замороженные продукты | пельмени, овощные смеси, наггетсы |
| `соусы` | Соусы, пасты, заправки | кетчуп, майонез, соевый соус, томатная паста |
| `овощи` | Свежие, иногда упакованные овощи | картошка, морковь, огурцы |
| `фрукты` | Фрукты, ягоды, цитрусы | бананы, яблоки, апельсины |
| `хлеб` | Хлеб, батоны, лаваш, хлебцы | батон, бородинский, тостовый хлеб |
| `чай` | Чай и чайные напитки | чай в пакетиках, листовой чай |
| `кофе` | Кофе и кофейные смеси | молотый кофе, растворимый кофе |
| `уход` | Товары личного ухода | крем для рук, шампунь, мыло |
| `быт` | Бытовые и хозяйственные товары | губки, пакеты, средство для посуды |
| `прочее` | Только если категория реально неясна | редкие или непонятные товары |

---

## Как выбирать категорию

| Ситуация | Что ставить |
| --- | --- |
| Товар жидкий, но это молочный продукт | `молочка`, а не `напитки` |
| Товар сладкий, но замороженный | `заморозка`, если ты хранишь и воспринимаешь его как заморозку |
| Пельмени, вареники, наггетсы | `заморозка` |
| Сосиски, колбаса, бекон | `мясо` |
| Чай в пакетиках | `чай` |
| Кофе 3 в 1 | `кофе` |
| Томатная паста | `соусы` |
| Сливки | `молочка` |

## Категории магазинов

Поле `kind` у магазина лучше заполнять из этого списка.

| `kind` | Когда использовать | Примеры |
| --- | --- | --- |
| `supermarket` | Обычный продуктовый магазин | Пятёрочка, Магнит, Перекрёсток |
| `hypermarket` | Большой гипермаркет | Ашан, Лента, Глобус |
| `discounter` | Магазин с упором на дешёвые товары | Светофор, Доброцен |
| `convenience` | Небольшой магазин у дома | local shop, киоск, мини-маркет |
| `marketplace` | Онлайн-площадка | Ozon, Wildberries, Яндекс Маркет |
| `delivery` | Сервис доставки продуктов | Самокат, Яндекс Лавка, Купер |
| `pharmacy` | Аптека | Апрель, Ригла, Горздрав |
| `household` | Бытовые/хозтовары | Fix Price, Улыбка радуги |
| `specialty` | Узкоспециализированный магазин | чайный магазин, мясная лавка |
| `other` | Если не подходит ничего выше | редкие случаи |

## Единицы измерения

Используй только эти значения в базе.

| Значение | Что означает | Когда использовать |
| --- | --- | --- |
| `pcs` | штуки | яйца, йогурты, упаковки, пакетики чая |
| `g` | граммы | крупы, сладости, сыр, мясо, макароны |
| `kg` | килограммы | большие упаковки, развес, если так удобнее |
| `ml` | миллилитры | молоко, сок, соусы, жидкие продукты |
| `l` | литры | большие бутылки и канистры |

---

## Что ставить в `base_unit`

`base_unit` это в чём ты хочешь считать товар по жизни.

| Тип товара | Рекомендуемый `base_unit` | Пример |
| --- | --- | --- |
| Штучные продукты | `pcs` | яйца, йогурты, сырки |
| Крупы и сыпучее | `g` | рис, гречка, овсянка |
| Жидкости | `ml` | молоко, кефир, сок |
| Большие объёмы воды | `ml` или `l` | вода 5 л |
| Заморозка в пачках | `g` или `pcs` | пельмени 900 г, мороженое 1 шт |
| Чай в пакетиках | `pcs` | 25 пакетиков |

---

## Что ставить в `typical_pack_unit`

`typical_pack_unit` это в чём указана обычная упаковка товара.

| Товар | `typical_pack_size` | `typical_pack_unit` |
| --- | --- | --- |
| Яйца | `10` | `pcs` |
| Молоко | `930` | `ml` |
| Вафли | `400` | `g` |
| Чай 25 пакетиков | `25` | `pcs` |
| Рис 900 г | `900` | `g` |
| Вода 1.5 л | `1.5` | `l` |

## Простое правило, если сомневаешься

| Вопрос | Что делать |
| --- | --- |
| Не знаю категорию | выбери ближайшую из таблицы, не ставь `прочее` без нужды |
| Не знаю бренд | оставь пусто |
| Не знаю точную фасовку | запиши то, что видно на упаковке |
| Не знаю `base_unit` | подумай, в чём ты будешь считать расход в рецептах |
| Не знаю `typical_pack_unit` | поставь ту единицу, которая написана на упаковке |

---

## Главная идея

Твоя система не должна быть идеально академической.

Она должна быть:

1. единообразной;
2. достаточно точной для покупок и рецептов;
3. удобной для ручного ведения.
