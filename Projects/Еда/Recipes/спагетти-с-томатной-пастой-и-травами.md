# Спагетти с томатной пастой и травами

## Ингредиенты

| Продукт | Кол-во | Ед. | Примечание |
| ------- | ------ | --- | ---------- |
| [[Спагетти 1кг]] | 180 | g | на 2 порции |
| [[Томатная паста кубаночка 800г]] | 70 | g | развести водой |
| [[Подсолнечное масло Щедрый год 1л]] | 15 | ml | |
| [[Чеснок сушеный 30г]] | 3 | g | |
| [[Базилик Каждый день 5 г]] | 1 | g | |
| [[Итальянские травы 15г Foodfest]] | 1 | g | |
| Вода | 150 | ml | для соуса |
| Соль | по вкусу |  | если есть |

## Шаги

1. Отвари спагетти до готовности.
2. На масле прогрей томатную пасту, добавь воду, чеснок и травы.
3. Провари соус 3-5 минут и смешай со спагетти.

## Заметки

> Базовый быстрый вариант, если не хочется долго готовить.
