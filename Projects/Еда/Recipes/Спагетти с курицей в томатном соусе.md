# Спагетти с курицей в томатном соусе

## Ингредиенты

| Продукт | Кол-во | Ед. | Примечание |
| ------- | ------ | --- | ---------- |
| Спагетти | 250 | г | Можно заменить другими макаронными изделиями |
| Куриные бедра | 2 | шт. | Отделить филе от кости, кожи и жира |
| Измельченные помидоры или томатный сок | 200 | мл | Можно заменить томатной пастой, разведенной водой |
| Чеснок | 1 | зубчик | Мелко порубить или пропустить через пресс |
| Лук репчатый | 1 | шт. | Нарезать кубиками |
| Сыр твёрдый | 30 | г | Натереть для подачи |
| Масло растительное | 2–3 | ст. л. | Лучше оливковое |
| Сахар | 1 | ч. л. | По желанию |
| Соль |  | по вкусу |  |
| Перец свежемолотый |  | по вкусу |  |
| Базилик сушеный |  | по вкусу |  |
| Специи |  | по вкусу | На выбор |
| Свежая зелень | 1 | веточка | Для подачи |

## Шаги

1. Подготовьте продукты. Спагетти можно заменить любыми макаронными изделиями.
2. Отварите спагетти в подсоленной кипящей воде согласно инструкции на упаковке.
3. Откиньте спагетти на дуршлаг и дайте воде стечь.
4. Пока варятся спагетти, отделите куриное филе от кости, кожи и жира. Нарежьте небольшими кусочками.
5. Лук очистите и нарежьте кубиками.
6. Разогрейте растительное масло на сковороде и обжарьте лук 2–3 минуты.
7. Добавьте к луку куриное филе. Поперчите и добавьте специи по вкусу.
8. Перемешайте и жарьте пару минут, пока курица не побелеет.
9. Добавьте измельченные помидоры или томатный сок, соль и сахар по желанию. Тушите около 10 минут на слабом огне.
10. Добавьте измельченный чеснок и сушеный базилик. Тушите еще около 5 минут.
11. Выложите сваренные спагетти в томатный соус с курицей.
12. Перемешайте и прогрейте несколько минут.
13. Подавайте, посыпав тертым сыром и украсив свежей зеленью.

## Заметки

> Сытное и простое блюдо для семейного или дружеского ужина.
