# Овсяная каша на воде

## Ингредиенты

| Продукт | Кол-во | Ед. | Примечание |
| ------- | ------ | --- | ---------- |
| [[Овсяные хлопья]] | 60 | g | |
| [[Сахар]] | 10 | g | по желанию |
| Вода | 250 | ml | |
| Соль | щепотка |  | если есть |

## Шаги

1. Доведи воду до кипения.
2. Засыпь хлопья и вари до нужной густоты.
3. Добавь сахар в конце.

## Заметки

> Самый базовый завтрак из того, что уже есть в базе.
