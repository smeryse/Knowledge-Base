# Обзор задач

Дашборд всех активных задач системы.

---

## 📅 Сегодня — активные

```dataview
TASK
FROM "Tasks/Daily"
WHERE !completed AND file.name = dateformat(date(today), "yyyy-MM-dd")
SORT text ASC
```

---

## 📥 Входящие (Waiting)

```dataview
TASK
FROM "Tasks/Waiting"
WHERE !completed
SORT text ASC
```

---

## 📅 Все невыполненные по дням

```dataview
TASK
FROM "Tasks/Daily"
WHERE !completed
GROUP BY file.link
SORT file.name DESC
```

---

## ✅ Выполненные сегодня

```dataview
TASK
FROM "Tasks/Daily"
WHERE completed AND completion = date(today)
SORT text ASC
```

---

## 🔥 Просроченные (не в сегодняшнем дне)

```dataview
TASK
FROM "Tasks/Daily"
WHERE !completed AND file.name != dateformat(date(today), "yyyy-MM-dd")
GROUP BY file.link
SORT file.name ASC
```
