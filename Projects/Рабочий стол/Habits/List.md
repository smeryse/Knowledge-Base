# Список привычек

Базовый набор ежедневных привычек. Каждая строка ниже автоматически копируется в новую daily note через шаблон `[[Projects/Рабочий стол/Templates/Daily Note|Daily Note]]`.

Редактируй этот список — шаблон подхватит изменения на следующий день.

## Активные привычки

- [ ] Выпить 4 стакана воды (+10)
- [ ] No Fap (+25)
- [ ] No Smoke (+5)
- [ ] No Sugar (+15)

---

## Как добавить новую

1. Отредактируй список выше.
2. Формат: `- [ ] Название (+N)`
3. Баллы: мелкие +5–+10, средние +15–+20.
