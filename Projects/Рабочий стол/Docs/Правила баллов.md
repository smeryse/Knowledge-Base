# Правила простановки баллов

См. актуальную версию: `[[Tasks/.qwen-task-rules.md]]`

## Краткая сводка

| Тип задачи | Диапазон | Примеры |
|------------|----------|---------|
| **Рутина** | +5 – +15 | Сменить постельное, закупиться |
| **Учебные** | +15 – +50 | Разбор задач, сдача лаб |
| **Технические** | +10 – +25 | Установка ПО, настройка |
| **Проекты** | +20 – +50 | Начало кода, бот |
| **Организация** | +10 – +25 | Структура, индексация |
| **Работа/заработок** | по сумме | Ozon Fresh, фриланс |

### Формат
```markdown
- [ ] Задача (+N)
- [ ] Задача в рублях (+1000р)  ← 5% = 50 баллов
```

### Запреты
- НЕ менять `[ ]` на `[x]`
- НЕ добавлять `✅`
- НЕ трогать уже выполненные
