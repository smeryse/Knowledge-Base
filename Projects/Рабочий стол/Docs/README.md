# README — Рабочий стол

Система управления задачами внутри Obsidian-vault. Вместо отдельного приложения — всё в тексте, ссылках и Dataview.

---

## Философия

1. **Одна задача = одна строка** `- [ ] задача (+N)`
2. **Баллы = мотивация** — каждая задача оценивается в баллах
3. **Ежедневный файл = контейнер дня** — задачи живут в `Tasks/Daily/YYYY-MM-DD.md`
4. **Waiting = inbox** — всё что не на сегодня, попадает в `Tasks/Waiting.md`
5. **Перенос без потерь** — `Ctrl+Shift+Enter` переносит задачу на завтра или в waiting

---

## Структура

```
Tasks/
  Daily/
    YYYY-MM-DD.md      ← ежедневная заметка
  Waiting.md           ← inbox / отложенные
  .qwen-task-rules.md  ← правила простановки баллов (AI)
```

---

## Формат задачи

```markdown
- [ ] Название задачи (+25)
- [ ] Купить молоко (+5)
- [ ] Заработать 1000р (+1000р)   ← 5% = 50 баллов
- [x] Выполненная задача (+25) ✅ 2026-05-12
```

---

## Геймификация

- `+N` — обычные баллы
- `+Nр` — рублёвые задачи (конвертируются в баллы с курсом 5%)
- Прогресс-бар считает `выполнено / общее` каждый день

---

## Команды

| Действие | Хоткей |
|---|---|
| Перенести задачу | `Ctrl+Shift+Enter` |
| Новая daily note | `Ctrl+D` (Templater) |
| Проставить баллы | команда AI |

---

## Интеграции

- **Dataview** — прогресс-бар, таблицы задач
- **Templater** — генерация daily notes с расписанием
- **Custom plugin** — postpone-task для переноса
