Отлично! Вот пошаговая инструкция по сборке системы автополива из видео:

## 📦 **Необходимые компоненты:**

1. **Digispark (ATtiny85)** — микроконтроллер
2. **MOSFET модуль** — для управления помпой
3. **Мини-помпа 3-6В** (водонасос)
4. **Батарейный отсек** на 3-4 батарейки АА
5. **Силиконовый шланг**
6. **Тройники/разветвители** для шлангов (опционально)
7. **Провода, паяльник**

## 🔌 **Схема подключения:**

**Digispark → MOSFET модуль:**
- Pin 0 (P0) → вход SIG на MOSFET
- GND → GND на MOSFET
- VCC (5V) → VCC на MOSFET (если помпа 5В)

**MOSFET → Помпа:**
- Выход MOSFET → плюс помпы
- GND → минус помпы
- Плюс питания → плюс помпы (параллельно)

**Питание:**
- Батарейный отсек → Digispark (5V и GND)

## 💻 **Прошивка:**

1. **Установи Arduino IDE** и добавь поддержку Digispark:
   - File → Preferences → Additional Boards Manager URLs
   - Добавь: `http://digistump.com/package_digistump_index.json`
   - Tools → Board → Boards Manager → найди "Digistump AVR" → установи

2. **Скачай прошивку** с https://alexgyver.ru/pump-sleep/

3. **В коде измени время:**
   ```cpp
   // Пример: полив каждые 3 дня
   int sleepTime = 60*60*24*3; // 259200 секунд
   ```

4. **Загрузи прошивку:**
   - Подключи Digispark **после** нажатия кнопки Upload
   - У тебя есть 5 секунд на подключение

## ⚙️ **Сборка:**

1. Соедини все компоненты по схеме
2. Помести помпу в ёмкость с водой
3. Шланг направь к растению
4. Запитай от батареек
5. Протестируй — помпа должна включаться на заданное время

## 🔧 **Важные моменты:**

- **Время работы помпы** обычно 10-30 секунд
- **Интервал полива** выставляй в секундах: 
  - 1 день = 86400 сек
  - 3 дня = 259200 сек
  - 7 дней = 604800 сек
- **Питание:** 3-4 батарейки АА обеспечат работу на несколько недель/месяцев
- **Защита от влаги:** закрой электронику от брызг

## ⚠️ **Возможные проблемы:**

- **Digispark не определяется** — установи драйверы
- **Помпа не качает** — проверь полярность и питание
- **Греется плата** — проверь соединения, возможно короткое замыкание

Нужна помощь с конкретным этапом?
Понял, это видео AlexGyver про простую систему автополива с помпой. Если тебе нужен **распылитель** (форсунка для мелкого распыления/тумана), то модификация несложная:

**Что изменится:**

1. **Насос** — нужен более мощный, так как для распыления требуется давление. Обычные мини-помпы 3-6V могут не справиться. Рекомендую:
   - Насос с давлением от 0.2-0.3 МПа
   - Или мембранный насос (diaphragm pump)

2. **Форсунка** — специальная распыляющая:
   - Форсунки для систем туманообразования
   - Форсунки от стеклоомывателей авто
   - Капельные эмиттеры с мелким распылением

3. **Важный момент** — распылители требуют **большего давления**, чем простой капельный полив. Если насос слабый, распыления не получится.

**Альтернативный вариант** — ультразвуковой генератор тумана (ultrasonic mist maker). Это отдельные модули на 24V, которые создают холодный туман, но им нужна постоянная подача воды в резервуар.

Для зелени на подоконнике какой вариант рассматриваешь — распыление воды на листья (туман) или полив под корень с распылителем?
