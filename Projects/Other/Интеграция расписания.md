# Интеграция расписания в Obsidian

## Описание

Автоматическое отображение расписания занятий в ежедневных заметках Obsidian с учётом чередования недель (числитель/знаменатель).

---

## Как это работает

### Определение недели

Скрипт автоматически вычисляет тип недели на основе даты:

- **Дата начала семестра:** 02.02.2026
- **Неделя I (числитель):** нечётные недели от начала семестра (1, 3, 5...)
- **Неделя II (знаменатель):** чётные недели от начала семестра (2, 4, 6...)

**Пример:**
- 02.02.2026 – 08.02.2026 → неделя 1 → I (числитель)
- 09.02.2026 – 15.02.2026 → неделя 2 → II (знаменатель)

### Получение расписания

При создании ежедневной заметки:

1. Templater извлекает дату из имени файла (формат `YYYY-MM-DD`)
2. Скрипт определяет день недели и тип недели
3. Из `schedule.json` загружаются пары на этот день
4. Фильтруются по типу недели (если пара только для I или II)
5. Вставляется таблица с расписанием

## Файлы

### schedule.json

Хранит расписание в формате JSON. Каждая пара содержит:

- `time` — время проведения
- `type` — тип занятия (ЛК, ЛР, ПР)
- `subject` — название предмета
- `teacher` — преподаватель
- `room` — аудитория
- `week` — неделя проведения (`null` = каждую неделю, `"I"` = числитель, `"II"` = знаменатель)

### schedule.js

Templater-скрипт на JavaScript:

- Определяет день недели из даты
- Вычисляет номер недели от начала семестра
- Возвращает тип недели (I или II)
- Фильтрует пары по неделе
- Формирует таблицу для вставки

### Daily-Note.md

Шаблон ежедневной заметки. Содержит секцию:

```markdown
## Расписание

<%* const schedule = await tp.file.include("Templates/scripts/schedule.js"); tR += schedule %>
```

## Использование

1. Открой календарь слева
2. Кликни на дату (или создай заметку вручную)
3. Заметка создастся с автоматически вставленным расписанием

---

## Настройка

### Изменить дату начала семестра

В `Templates/scripts/schedule.js` измени:

```javascript
const semesterStart = new Date("2026-02-02"); // новая дата
```

### Добавить/изменить пары

Отредактируй `Study/schedule.json`:

1. Найди нужный день (1 = понедельник, ..., 6 = суббота, 0 = воскресенье)
2. Добавь пару в массив `classes`
3. Укажи `week: null` для еженедельных или `"I"`/`"II"` для чередующихся
