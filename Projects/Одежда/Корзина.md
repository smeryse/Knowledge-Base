# 🧺 Корзина грязного белья

> [!INFO] Как это работает
> - Когда вещь достигает лимита носки, она автоматически попадает в корзину
> - **Чёрная корзина:** тёмные вещи (чёрные, тёмно-синие, бордовые)
> - **Белая корзина:** светлые вещи (белые, бежевые, светло-серые)
> - Лимит: **4.8 кг** (80% от 6 кг) — после этого пора стирать

---

## ⚫ Чёрная корзина

```dataviewjs
const LAUNDRY_LIMIT = 4800;
const ITEMS_DIR = 'Projects/Одежда/items';

// Получаем все markdown файлы из папки items
const allFiles = app.vault.getMarkdownFiles();
const itemFiles = allFiles.filter(f => f.path.startsWith(ITEMS_DIR));

const blackItems = [];
let totalWeight = 0;

for (const file of itemFiles) {
    const content = await app.vault.read(file);
    const yamlMatch = content.match(/^---\n([\s\S]*?)\n---/);
    if (!yamlMatch) continue;
    
    const yamlStr = yamlMatch[1];
    const yaml = {};
    
    for (const line of yamlStr.split('\n')) {
        const match = line.match(/^(\w+):\s*(.+)$/);
        if (match) {
            let key = match[1];
            let value = match[2].trim();
            if (value.startsWith('"') && value.endsWith('"')) value = value.slice(1, -1);
            if (value === 'true') value = true;
            else if (value === 'false') value = false;
            else if (value === 'null') value = null;
            else if (!isNaN(value) && value !== '') value = Number(value);
            yaml[key] = value;
        }
    }
    
    if (yaml.is_dirty && yaml.laundry_basket === 'black') {
        blackItems.push(yaml);
        totalWeight += yaml.weight || 0;
    }
}

// Прогресс-бар
const percent = Math.round((totalWeight / LAUNDRY_LIMIT) * 100);
const width = 40;
const filled = Math.min(Math.round((totalWeight / LAUNDRY_LIMIT) * width), width);
const bar = '█'.repeat(filled) + '░'.repeat(width - filled);

dv.paragraph(`**Вес:** \`${bar}\` **${(totalWeight / 1000).toFixed(2)} кг / 4.8 кг** (${percent}%)`);

if (blackItems.length === 0) {
    dv.paragraph("*Корзина пуста* ✨");
} else {
    dv.table(["Вещь", "Цвет", "Вес", "Надето"], 
        blackItems.map(item => [
            item.name,
            item.color,
            item.weight_str || `${item.weight} г`,
            `${item.wears_count}/${item.wash_after_wears}`
        ])
    );
}
```

## 🔄 После стирки

Открой файл вещи в `items/` и вручную сбрось:
```yaml
is_dirty: false
wears_count: 0
laundry_basket: null
```

> Потом добавим кнопку для автоматического сброса корзины.
